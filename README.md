## Overview

This is an interactive JavaScript quiz app that displays questions one at a time, lets the user select an answer, gives instant feedback, tracks score, and shows a final result at the end — including a full answer review.

---

## Features

- Dynamic questions loaded from a JavaScript array
- Multiple choice answers
- Instant feedback (correct/incorrect)
- Score tracking
- Progress display (Question X of Y)
- Restart quiz functionality
- Randomized question order
- Answer review at the end (see what you got right/wrong)
- Start screen before quiz begins

---

## Tech Used

- HTML
- CSS
- JavaScript

---

## How It Works

1. Questions are stored in an array
2. The quiz starts from a start screen
3. Questions are shuffled randomly
4. A function loads each question onto the page
5. User selects an answer
6. The app checks if it’s correct and gives feedback
7. Score updates
8. User moves to the next question
9. Final score is shown
10. A full answer review is displayed

---

## How to Run

1. Download or clone the project
2. Open index.html in your browser
3. Start the quiz
